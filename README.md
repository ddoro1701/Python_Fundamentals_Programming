# Python_Fundamentals_Programming

This repository contains a collection of Jupyter Notebooks to practice **basic Python programming concepts**.

## Topics covered

- Lists and sorting (e.g. `list_sort.ipynb`, `reverse_python_list.ipynb`, `Sorted_List_of_Names.ipynb`)
- Dictionaries, sets and hashing (e.g. `DictionarySetHash.ipynb`)
- Queues and basic data structures (e.g. `Queue1.ipynb`)
- Loops, ranges and simple tasks (e.g. `Task_forNumberInRange.ipynb`)
- Functions and arrays/lists (e.g. `Function_Array_easy.ipynb`)
- Anagrams and simple algorithms (e.g. `Sorted_Anagram.ipynb`)
- Intro to linear regression with scikit-learn (e.g. `linear_regression_mit_sklearn.ipynb`)
- Small practice examples (e.g. `ducks_squares.ipynb`)
